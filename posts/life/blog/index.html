create a beatiful blog⌗

what I need?⌗

  • create a blog
  • easy to maintain
  • use markdown
  • minimalism

hugo⌗

hugo is a static site generator. Why choose hugo ? It’s the most popular static site generator in github. I just need one, the only requirement is easy to use, that means it won’t waste my time to config.

theme: terminal⌗

minimalist

get start⌗

the main steps are listed in documentation of hugo.

  • install hugo use port install hugo 
    port install hugo

    or we use brew, or download binary in realise page

    brew install hugo
  • create site hugo new site challen
  • install theme hint: If you want to update the theme, you can clone the repo, but I dont.
    • go to https://github.com/panr/hugo-theme-terminal/ download
    • extract zipfile and move it to theme foldermv hugo-theme-terminal-master theme/terminal
  • config site this config is copy from the theme terminal default config in README.
  • add first page hugo new helloworld.md
  • run the local server hugo server -D
  • generate the static file hugo -D
  • test static file cd public & python -m SimpleHTTPServer 3000
